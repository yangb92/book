screen分屏命令

screen

安装 screen，使用命令 apt-get install screen 即可完成。

screen 常用快捷键如下：

  • Ctrl+a 再按shift + s 水平分割当前窗口
  • Ctrl+a | 垂直分割当前窗口
  • Ctrl+a c 创建一个新的运行 shell 的窗口并切换到该窗口
  • Ctrl+a X 关闭当前窗口
  • Ctrl+a x 锁定当前窗口
  • Ctrl+a k 杀死当前窗口
